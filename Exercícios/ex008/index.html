Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> e </b> (forma não semântica - sem significado para o código).

Nesta frase temos um termo em negrito usando a tag <strong> e </strong> (forma semântica - com significado para o código).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> e </i> (forma não semântica - sem significado para o código).

Nesta frase temos um termo em ênfase usando a tag <em> e </em> (forma semântica - com significado para o código).

Texto marcado

Nesta frase temos um termo marcado usando a tag <mark> e </mark> (forma semântica - com significado para o código).

Texto grande e pequeno

Estamos criando nesse parágrafo um texto grande utiliando a tag <big> e </big> e um texto pequeno utilizando a tag <small> e </small> - OBS: A tag big já é obsoleta.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado utilizando a tag <del> e </del> (forma semântica - com significado para o código).

Texto inserido

Podemos marcar um texto inserido para dar uma ênfase e indicar que ele foi adicionado depois utilizando a tag <ins> e </ins> (forma semântica - com significado para o código).

Existe também o sublinhado com a tag u (forma não semântica - sem significado para o código).

Texto sobrescrito

Para inserir coisas do tipo x20+3 utilizamos a tag <sup> e </sup> (forma semântica - com significado para o código).

Texto subescrito

Para inserir coisas do tipo H2O utilizamos a tag <sub>(forma semântica - com significado para o código).

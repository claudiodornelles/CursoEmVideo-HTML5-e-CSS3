  • Links
    • Externos

      • Utiliza-se os atributos target="_blank" e rel="external" para que as ferramentas de busca entendam que o link está direcionando para fora da sua página e ainda, que essa página seja aberta em uma nova aba.

    • Internos

      • Utiliza-se os atributos target="_self" e rel="next / prev" para que as ferramentas de busca entendam que o link está direcionando para uma página seguinte ou anterior à pagina atual e ainda, que essa página seja aberta na mesma aba.

      Caso o link seja para retroceder para uma página que se encontra em outra pasta do repositório, devemos utilizar o início do caminho como "../"
